> # Zombie Game

자바 다형성 콘솔 게임 구현

<br>

### 게임 설명
> - 맵은 1~20의 칸 으로 구성되어 있습니다.
> - 플레이어는 한칸씩 이동 가능합니다.
> - Zombie 또는 BossZombie 를 만나면 전투 실행
> - 마지막 20번 칸에 도달하거나 플레이어가 죽으면 게임은 종료됩니다.
> - Boss와 Zombie 는 각각 4개, 2개의 스킬을 보유하고 있습니다.
